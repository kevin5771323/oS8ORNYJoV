# 爱相连儿童慈善管理系统 java989

## 项目概述

爱相连儿童慈善管理系统是基于SpringBoot和Vue的MVC架构模式，实现了前后端分离的技术架构。它旨在为儿童慈善机构提供一个高效、稳定的管理平台。

## 技术栈

- 后端：SpringBoot、MyBatis
- 前端：Vue
- 数据库：MySQL 8.0
- 开发环境：JDK 1.8、Maven、npm
- 开发工具：IDEA、Eclipse、VSCode
- 数据库可视化工具：Navicat

   - 用户注册、登录、权限管理
2. 慈善项目管理

3. 捐赠管理
   - 捐赠记录、捐赠者信息管理
4. 志愿者管理
